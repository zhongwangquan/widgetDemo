# MVUE

> A Vue.js Component


## Doc
[Page](https://github.com/jiakeqi/mvue/tree/master/src/components/page/README.md)

## Install
``` shell
npm install mvue -S
```
## Usage
``` JavaScript
import {Page} from 'mvue'
Vue.use(Page)
```
## Build
``` bash
# 安装依赖
npm install

# 运行项目 localhost:8080
npm run dev

# 打包 dist 文件
npm run prod

```
